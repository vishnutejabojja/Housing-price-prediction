# House-Price-Prediction

A project which predicts the house prices in Bangalore based on the input parameters.

## Dataset link:
https://www.kaggle.com/amitabhajoy/bengaluru-house-price-data

## Technology used:
  
  * Python
  * Flask
  * Javascript
  * Html
  * CSS
  
## Algorithm used:
  Linear Regression
